# odoo_learning_mcp
This repository is used for learning how MCP works with Odoo 18

## Prerequisites
1. Install Odoo 18 (Sale Module must be installed with invoice transaction)
2. Install FastMCP

## Steps to Run
1. Run the Odoo Server, Odoo MCP client on the same port, e.g. 8069
2. Run the MCP Server on different port, e.g. 8000
3. Provide input with the SO invoice context e.g. 'Tell me the SO invoice that is not yet done'
